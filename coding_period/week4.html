Coding Period — Week 4 (22 June – 28 June)

Implemented the KML Upload Module — POST /api/tours/upload.

Delivered:

  • Multer storage under /uploads with size limits.

  • Duplicate filename detection and basic metadata extraction (name, size).
  • Auth-gated upload so anonymous clients cannot write files.

Marc suggested logging every upload attempt (success and failure) for museum operators. Next: tour organization APIs (rename, move, tag, favorite).
