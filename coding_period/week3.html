Liquid Galaxy Tour Management Server

Tue, June 21

Coding Period — Week 3 (15 June – 21 June)

Built the data layer for tours and activity logs so the server can catalog hundreds of KML files.

Delivered:

  • Tours collection: title, description, filename, filepath, category, tags, uploadedBy, createdAt.
  • Logs collection: user, action, status, timestamp, ipAddress.
  • Mongoose indexes on title, tags, and uploadedBy for faster search.
  • Service helpers for create / update / soft-delete tour metadata.
