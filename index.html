Liquid Galaxy Tour Management Server

Mon, September 12

Hi — I’m Aditya Shukla. This is the GSoC project blog for Liquid Galaxy Tour Management Server under the Liquid Galaxy Project. Mentor: Marc Gonzalez Capdevila.

Liquid Galaxy is a multi-display visualization platform powered by Google Earth. This project builds a Node.js / Express backend that uploads, organizes, and launches KML tours on the cluster through REST APIs, Socket.IO, and secure SSH — so operators no longer need manual file transfers and shell commands.
